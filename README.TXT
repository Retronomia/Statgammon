========================
      HOW TO PLAY
========================
Goal: get both pieces off of the board
-The first piece is "1"
-The second piece is "2"
-When pieces are at the same position, the value is shown as "3"
-Empty spaces are "0"
-To get the piece off of the board, the number on the die must be the exact value needed
-Numerical responses are typed in as input when prompted
========================
   RUNNING THE FILE
========================
In order to play Statgammon, keep all of the files inside of the Statgammon folder.
Running the exe file by clicking it is the easiest method.
